# Конвертер WebM/Ogg/MP4 в MP4/MP3

Графический интерфейс для конвертации видеофайлов WebM, аудиофайлов Ogg и видеофайлов MP4 в форматы MP4 и MP3 с поддержкой пакетной обработки.

## Структура проекта

```
ffmpeg-converter/
├── ffmpeg_converter.py          # Основной файл с реализацией конвертера
├── README.md                   # Основное описание проекта
├── requirements.txt            # Зависимости проекта
├── .gitignore                  # Файл игнорирования для git

```

## Основные функции

1. **Графический интерфейс** на базе Tkinter
2. **Поддержка форматов**:
   - WebM → MP4
   - OGG → MP3
   - MP4 → MP4 (копирование)
3. **Пакетная обработка** файлов
4. **Отображение прогресса** конвертации
5. **Обработка ошибок** и уведомления
6. **Поддержка выбора папок и файлов**
7. **Выбор битрейта** при конвертации в MP3

## Технологии

- **Python 3.x** - основной язык программирования
- **Tkinter** - графическая библиотека для создания интерфейса
- **FFmpeg** - внешняя программа для конвертации медиафайлов
- **subprocess** - для вызова FFmpeg из Python

## Требования

- Python 3.x
- FFmpeg (должен быть установлен и добавлен в PATH)

## Установка

1. Установите Python 3.x с официального сайта (https://www.python.org/downloads/)

2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   ```
3. Активируйте виртуальное окружение:
   - source venv/Scripts/activate # Windows
   - source venv/bin/activate # Linux/macOS: 

4. Установите зависимости (при необходимости):
   ```bash
   pip install -r requirements.txt
   ```
5. Установите FFmpeg:
   - Windows: Скачайте с https://ffmpeg.org/download.html и добавьте в PATH
   - Linux: `sudo apt install ffmpeg`
   - macOS: `brew install ffmpeg`

## Использование

1. Запустите приложение:
   ```bash
   python ffmpeg_converter.py
   ```

2. Добавьте файлы для конвертации:
   - Нажмите "Добавить файлы" для выбора отдельных файлов
   - Нажмите "Добавить папку" для выбора всех файлов в папке

3. Выберите формат конвертации:
   - MP4 для WebM файлов
   - MP3 для OGG файлов

4. При конвертации в MP3 можно выбрать битрейт из доступных значений

5. Нажмите "Начать конвертацию"
